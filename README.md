# 📚 Study Tracker Dashboard v2

A modern SaaS-style Study Tracker Dashboard built using HTML, CSS, and JavaScript.

Track your study sessions, monitor learning progress, maintain study streaks, visualize activity using a heatmap, and achieve your daily study goals through a clean and professional dashboard.

---

## 🚀 Features

### 📖 Study Session Management

* Add study sessions
* Delete study sessions
* Track subjects studied
* Record study hours

### 📊 Analytics Dashboard

* Total Subjects
* Total Study Hours
* Current Study Streak
* Goal Completion Percentage

### 🎯 Goal Tracking

* Set Daily Study Goals
* Track Goal Progress
* Visual Progress Bar

### 🔥 Activity Heatmap

* GitHub-style activity visualization
* Track study consistency
* Visual learning streaks

### 📜 Study History

* View previous study sessions
* Chronological learning records
* Activity timeline

### 💾 Local Storage

* Saves all study sessions
* Persists goals
* Retains activity history
* No database required

### 📱 Responsive Design

* Desktop Friendly
* Tablet Friendly
* Mobile Friendly
